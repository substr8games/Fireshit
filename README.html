Handcrafted Wayfire utility suite

Fireshit

An independently handcrafted set of tools for Wayfire Compositor on Arch Linux: persistent passive and active monitoring, low-latency capture, and broad settings accessibility across your userspace.

May or may not work on other distributions.

Persistent telemetry surface

SUBSTR8 HUD

A passthrough Wayfire overlay that keeps terminal telemetry visible without forcing you to choose between monitoring and using the desktop.

Suite component
  • Provides three terminal surfaces in a passthrough overlay selected with SUPER+Home.
  • Delivers live telemetry at a glance through a deliberately active visual interface.
  • Stays above the desktop and behind ordinary windows in its default state.
  • Cycles through desktop, top, interactive, and invisible states with SUPER+Home.
  • Integrates htop, btop, iotop, nvtop, nethogs, and OpenSnitch telemetry.
  • Mirrors the real TTY running Wayfire in a dedicated read-only panel.
Targeted Wayland capture

FrameTap

Captures an exact application, a complete output, or a selected region and produces a shareable viewport, an MKV recording, a screenshot, or the appropriate combination.

Suite component
  • Captures an application window, an entire screen, or a custom region.
  • Creates a live shareable viewport, an .mkv recording, or both.
  • Launches applications directly with frametap [options] -- <application> [args...].
  • Records only the launched application’s audio while leaving microphones and unrelated applications untouched.
  • Records the system output mix during region or full-screen capture.
  • Selects targets by focused window, application ID, view ID, complete output, or region.
  • Uses a direct compositor-to-encoder path designed to minimize copying and added latency.
  • Takes standard full-screen screenshots and interactive screen snips.
Fireshit core

Firemod

The central suite authority: it tracks available tools, discovers attributable configuration, and projects registered actions into a fast pointer-centered utility grid.

Suite component
  • Unified Module Registry: dispatches global actions to registered first-party and external capabilities.
  • Curated Settings Discovery: recognizes INI, TOML, JSON, YAML, and related sources while filtering ambiguous results.
  • Draft & Revision Safety: keeps edits deliberate and preserves up to ten historical recovery states.
  • Flat-Top Hex Grid: maps system actions to stable, muscle-memory positions around the pointer.
  • Unified Super Bind: uses SUPER+middle-click to summon, relocate, or dismiss the utility surface.
  • Integrated Color Picker: samples pixels through a local 9×9 magnifier and copies practical color formats.
