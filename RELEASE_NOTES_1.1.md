## Corrected in this release

- Replaced first-party MIT declarations with `LicenseRef-FCIL-0.1`.
- Aligned Firemod, FrameTap, and SUBSTR8 HUD package metadata to suite version 1.1.
- Added the canonical root `run.sh` dispatcher required by the public documentation.
- Restored all public documentation images.
- Added concrete private vulnerability-reporting instructions.
- Completed the release-specific third-party material declaration.
- Removed Firestate executable claims from Firemod metadata until those binaries are actually included.
- Narrowed FCIL competitive-capture language so permitted noncommercial forks remain permitted.
- Added an interoperability allowance for minimal interface declarations, schemas, constants, and stubs.

- Removed stale 1.0 prebuilt binaries rather than falsely relabeling them as 1.1.
